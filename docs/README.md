# Melos Documentation Index

This directory keeps architecture and research notes small and non-overlapping.

## Architecture

- `architecture/api-boundaries.md`: package responsibilities, dependency
  direction, public API layering.
- `architecture/rendering-pipeline.md`: renderer pipeline, spacing model,
  glyph-plan direction, renderer migration order.

## Research

- `research/latex-music-rendering.md`: source-code lessons from MusiXTeX, PMX,
  M-Tx, LilyPond, LyLuaTeX, and Gregorio.
- `research/notation-api-comparison.md`: source-level API comparison with
  VexFlow, OpenSheetMusicDisplay, abcjs, and LilyPond.
- `research/notation-capability-matrix.md`: current notation capability status,
  priority gaps, and fixture targets.

Keep implementation plans in architecture docs. Keep source evidence and
external comparisons in research docs.
