# Test Data

This directory contains test data used for validating the `melos` converter.

## Sources

The data in the `musicxml` and `mnx-groundtruth` subdirectories is sourced from the **[W3C MNX Converter](https://github.com/w3c/mnxconverter)** project.

*   **`musicxml/`**: Contains `.musicxml` files used as input for conversion tests.
*   **`mnx-groundtruth/`**: Contains official `.mnx` files used as the expected output (ground truth) for validation.
